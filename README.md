# LLM Zoomcamp 2026

My notes, code, and homework solutions for [LLM Zoomcamp](https://github.com/DataTalksClub/llm-zoomcamp) by [DataTalks.Club](https://datatalks.club) — a free course on building practical LLM applications: RAG, vector search, agents, evaluation, monitoring, and more.

## Setup

- Package management: [`uv`](https://github.com/astral-sh/uv)
- LLM: [Ollama](https://ollama.com) (`gemma4:31b-cloud`)
- Each module lives in its own folder with its own `uv` project (`pyproject.toml`, `uv.lock`)
